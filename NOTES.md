### NOTES

- Set `zathura` as default PDF Viewer.

```toml
# Saved :: $HOME/.config/mimeapps.list

[Default Applications]
application/pdf=org.pwmt.zathura.desktop
```

- Use `asdf` to manage tools.

```toml
# Saved :: $HOME/.tool-versions

nodejs 20.0.0
golang 1.22.0
opam 2.1.6
ruby 3.3.0
rust 1.78.0
php 8.3.7
bun 1.1.10
```
